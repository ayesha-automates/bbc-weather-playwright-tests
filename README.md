
# BBC Weather Playwright Tests

A Playwright automation project written in JavaScript for testing BBC Weather.

## Tests Included

- Manchester weather forecast test
- Multiple-city weather tests
- Invalid location negative test
- Temperature and weather summary checks
- Cross-browser testing

## Browsers

- Chromium
- Firefox
- WebKit

## Run Tests

npx playwright test

## View Test Report
npx playwright show-report

## CI Testing

GitHub Actions automatically runs the tests whenever new code is pushed.
